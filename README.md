# eviebot
small bot made for 2 servers i own.

using [serenity](https://github.com/serenity-rs/serenity) library to interact with discord api.

[serenity](https://github.com/serenity-rs/serenity) is licensed under the [ISC License](https://github.com/serenity-rs/serenity/blob/current/LICENSE.md).

if you're in one of these servers, and are reading this, go to the issues tab and request some features.
